Présentation du projet

JavaStock est une application de gestion de stock développée en Java avec une interface graphique JavaFX. Conçue dans le cadre du BTS SIO SLAM, elle permet à une entreprise de gérer son inventaire de manière complète et intuitive depuis une application desktop.

L'application suit le patron de conception MVC (Modèle-Vue-Contrôleur) pour séparer la logique métier de l'interface utilisateur, garantissant une architecture maintenable et extensible.

Architecture

Patron MVC

Le projet est structuré selon le patron MVC, une bonne pratique fondamentale en développement logiciel.

Model

Classes Java représentant les entités métier (Produit, Categorie, Mouvement) et les interactions avec la base de données MySQL.

View

Fichiers FXML définissant les interfaces graphiques JavaFX — tableaux, formulaires et graphiques de stock.

Controller

Classes Java gérant les interactions utilisateur, les validations et la logique de navigation entre les vues.

DAO

Couche d'accès aux données avec des requêtes préparées JDBC pour sécuriser les interactions avec MySQL.

Fonctionnalités

Ce que l'application fait

Gestion des produits

Ajout, modification, suppression et recherche de produits avec catégories et référence unique.

Entrées / Sorties de stock

Enregistrement des mouvements de stock avec historique complet et mise à jour automatique des quantités.

Tableau de bord

Vue synthétique avec graphiques JavaFX Charts — stock par catégorie, alertes de rupture, évolution des mouvements.

Alertes de stock bas

Notifications automatiques lorsqu'un produit atteint un seuil de stock minimum configurable.

Export CSV

Export de l'inventaire complet ou filtré au format CSV pour traitement externe (Excel, etc.).

Recherche et filtres avancés

Recherche par nom, référence ou catégorie avec filtres sur le statut du stock en temps réel.

Ce que j'ai appris

Ce projet m'a permis d'approfondir la programmation orientée objet en Java et la conception d'interfaces graphiques avec JavaFX. L'implémentation du patron MVC m'a enseigné comment structurer une application de manière professionnelle et maintenable.

La gestion de la connexion JDBC et la mise en place de requêtes préparées pour sécuriser les accès à la base de données MySQL ont été particulièrement formatrices. J'ai également appris à gérer les exceptions et à concevoir une expérience utilisateur cohérente dans une application desktop.
